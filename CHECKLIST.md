## 📋 Что нужно сделать прямо сейчас

1. **Сгенерируйте JWT секрет:**
   ```bash
   node generate-secret.js
   ```

2. **Создайте App Password для Gmail:**
   - Войдите в Google Account
   - Включите 2FA
   - Создайте App Password для "Mail"

3. **Загрузите проект в GitHub:**
   ```bash
   git init
   git add .
   git commit -m "Ready for Railway deployment"
   git remote add origin https://github.com/ваш-username/fegr-messenger.git
   git branch -M main
   git push -u origin main
   ```

4. **Перейдите на Railway.app и:**
   - Создайте новый проект
   - Подключите GitHub репозиторий
   - Настройте переменные окружения
   - Добавьте MongoDB
   - Настройте домен fegrem.ru

## 🎯 Цель

После выполнения всех шагов ваш Fegr Messenger будет доступен по адресу:
- **Основной домен:** https://fegrem.ru
- **www перенаправление:** https://www.fegrem.ru → https://fegrem.ru
- **Полная функциональность:** ✅ HTTPS, ✅ Чаты, ✅ Посты, ✅ Email верификация
