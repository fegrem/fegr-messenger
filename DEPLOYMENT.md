# 🚀 Развертывание Fegr Messenger на Railway

### 1. Требования

- [x] Аккаунт на [GitHub](https://github.com)
- [x] Аккаунт на [Railway.app](https://railway.app)

## Настройка Railway

### 1. Подключение репозитория

1. Зайдите на [Railway.app](https://railway.app)
2. Нажмите "New Project"
3. Выберите "Deploy from GitHub repo"
4. Выберите ваш репозиторий `fegr-messenger`
5. Нажмите "Deploy"

### 2. Настройка переменных окружения

Перейдите в **Settings → Variables** и добавьте:

```bash
# Обязательные переменные
JWT_SECRET=ваш-сильный-секрет-ключ-минимум-32-символа-лучше-использовать-генератор
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=ваш-email@gmail.com
SMTP_PASS=ваш-app-password-от-gmail

# Опциональные (Railway установит автоматически)
# PORT=5000
# NODE_ENV=production
# DATABASE_URL=mongodb://...
```

**⚠️ Важно:** Для Gmail нужно создать App Password:
1. Войдите в Google Account Settings
2. Включите двухфакторную аутентификацию
3. Перейдите в Security → App passwords
4. Создайте пароль для приложения "Mail"

### 3. Настройка MongoDB

Railway автоматически создаст MongoDB базу данных:

1. В Railway перейдите в "Add" → "Database"
2. Выберите "MongoDB"
3. Railway создаст DATABASE_URL переменную автоматически
4. База данных будет доступна по этой переменной

## Настройка домена fegrem.ru

### 1. В Railway

1. Перейдите в **Settings → Domains**
2. Нажмите "Add Domain"
3. Введите `fegrem.ru`
4. Railway покажет DNS-записи для настройки

### 2. В панели управления доменом

Добавьте DNS записи (пример для Cloudflare):

```
Тип: CNAME
Имя: @ (или fegrem.ru)
Значение: ваш-проект.up.railway.app
TTL: 3600
Прокси: OFF
```

Или A запись (если Railway предоставит IP):
```
Тип: A
Имя: @
Значение: [IP-адрес от Railway]
TTL: 3600
```

### 3. Настройка www перенаправления

```
Тип: CNAME
Имя: www
Значение: fegrem.ru
TTL: 3600
Прокси: OFF
```

### 4. В Railway добавьте www домен

1. Добавьте домен `www.fegrem.ru`
2. Настройте перенаправление на `fegrem.ru`

## Возможные проблемы и решения

### 🔴 Домен не работает

**Причина:** DNS записи не распространились
**Решение:** Подождите 24 часа, проверьте DNS записи

**Причина:** Неправильные DNS записи
**Решение:** Проверьте тип записи (CNAME vs A), значение и TTL

### 🔴 Email не отправляется

**Причина:** Неправильный App Password
**Решение:** Пересоздайте App Password в Google Account

**Причина:** 2FA не включена
**Решение:** Включите двухфакторную аутентификацию в Google Account

### 🔴 MongoDB подключение не работает

**Причина:** DATABASE_URL не настроен
**Решение:** Убедитесь, что MongoDB добавлен в Railway

**Причина:** Неправильные настройки MongoDB
**Решение:** Проверьте настройки безопасности в MongoDB Atlas

### 🔴 Socket.IO не работает

**Причина:** CORS настройки
**Решение:** Проверьте allowedOrigins в server.js

**Причина:** HTTPS/HTTP несоответствие
**Решение:** Убедитесь, что сайт работает по HTTPS
